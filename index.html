FAQ'S

Q: What is JavaScript?
A: JavaScript is a programming language that is primarily used for client-side web development. It is a high-level, dynamic, and interpreted language that allows developers to add interactivity and dynamic behavior to web pages.
Q: How do I add JavaScript to my web page?
A: You can add JavaScript to your web page in a few ways. You can include the JavaScript code directly in your HTML file using a script tag. You can also link to an external JavaScript file using the script tag. Alternatively, you can use a JavaScript bundler like Webpack or Rollup to bundle your JavaScript files together and include them in your HTML file.
Q: What are the basic data types in JavaScript?
A: JavaScript has six primitive data types: string, number, boolean, null, undefined, and symbol. It also has one non-primitive data type, object.
Q: How do I declare a variable in JavaScript?
A: You can declare a variable in JavaScript using the var, let, or const keywords. The var keyword is used to declare a variable globally or function-scoped. The let keyword is used to declare a block-scoped variable. The const keyword is used to declare a block-scoped variable that cannot be reassigned.
Q: What is hoisting in JavaScript?
A: Hoisting is a mechanism in JavaScript that allows variables and function declarations to be moved to the top of their respective scopes. This means that you can use a variable or function before it is declared.
